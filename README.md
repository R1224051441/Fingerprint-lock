# 🔐 指纹锁项目

## 📌 项目简介

本项目是一个 **基于 STM32F103C8T6 的指纹锁系统**，使用 **HAL 库** 进行开发，实现了指纹识别、存储、状态指示以及外设扩展等功能，适合用于嵌入式学习与课程设计。

---

## 🧠 主控芯片

* **STM32F103C8T6**
* 开发环境：STM32CubeMX + Keil / VS Code（HAL 库）

---

## 🧩 功能模块

项目集成了多种常见外设模块，功能较为完整：

* 🔍 **指纹模块**：ZW101
  用于指纹录入、比对与验证

* 🖥️ **显示模块**：OLED
  显示系统状态、操作提示等信息

* 💾 **外部存储**：W25Q64
  用于存储指纹数据或系统配置

* 🌈 **状态指示灯**：WS2812
  通过 RGB 灯效显示不同系统状态

* 📶 **蓝牙模块**：HC-05
  支持蓝牙通信，可进行调试或远程控制

* 🎛️ **输入设备**：按键
  用于本地操作与模式切换

---

## ⚙️ 主要功能（可扩展）

* 指纹注册与删除
* 指纹比对开锁
* OLED 实时状态显示
* 蓝牙通信控制
* WS2812 状态指示
* 外部 Flash 数据存储
